Set w JavaScript to struktura danych, która przechowuje unikalne wartości, bez powtórzeń. Może zawierać różne typy danych, takie jak liczby, ciągi znaków, obiekty itp.

Podstawowe metody obiektu Set:

  • add(value): Dodaje wartość do zbioru.
  • delete(value): Usuwa wartość ze zbioru.
  • has(value): Sprawdza, czy wartość istnieje w zbiorze.
  • clear(): Usuwa wszystkie elementy ze zbioru.
  • size (właściwość): Zwraca liczbę elementów w zbiorze.
  • forEach(callbackFn): Wykonuje podaną funkcję zwrotną dla każdego elementu zbioru.
  • values(): Zwraca iterator z wartościami zbioru.
  • keys(): To samo co values(), ale jest dostępne dla zachowania zgodności z interfejsem iteracji.
  • entries(): Zwraca iterator z tablicami zawierającymi pary [value, value], gdzie pierwszy element to wartość, a drugi to również wartość (zgodność z Map).

Set jest użyteczny w przypadkach, gdy chcesz przechowywać jedynie unikalne wartości i nie zależy Ci na kolejności elementów. Może być stosowany do eliminowania duplikatów z tablicy, sprawdzania przynależności elementu do zbioru, wykonywania operacji na zbiorach itp.

Przykład użycia Set:

javascript
const mySet = new Set();
mySet.add(1);
mySet.add(2);
mySet.add(3);
mySet.add(2); // Duplikat, nie zostanie dodany

console.log(mySet.size); // 3

console.log(mySet.has(2)); // true

mySet.delete(3);
