Typy generyczne w TypeScript to mechanizm, który umożliwia tworzenie ogólnych i wielozadaniowych komponentów, funkcji i klas, które mogą operować na różnych typach danych. Dzięki typom generycznym w TypeScript, możemy tworzyć bardziej elastyczne i bezpieczne rozwiązania, które automatycznie dostosowują się do konkretnych typów przekazywanych danych.

Typy generyczne pozwalają na parametryzację typów, które są używane w definicji funkcji lub klasy. Dzięki temu możemy tworzyć funkcje lub klasy, które działają na różnych typach danych, jednocześnie zapewniając statyczną kontrolę typów.

Przykład użycia typów generycznych w TypeScript:

typescript
// Generyczna funkcja, która zwraca przekazany argument
        function identity<T>(arg: T): T {{'{'}}
          return arg;
        }
        
        // Wywołanie funkcji z argumentem typu string
        let result = identity<string>("Hello");
        console.log(result); // Wyświetli: Hello
        
        // Wywołanie funkcji z argumentem typu number
        result = identity<number>(42);
        console.log(result); // Wyświetli: 42

W tym przykładzie, generyczna funkcja identity przyjmuje argument arg o dowolnym typie T i zwraca wartość tego samego typu. Możemy określić konkretny typ podczas wywoływania funkcji, na przykład identity<string>("Hello"), co spowoduje, że funkcja zwróci wartość typu string.

Typy generyczne w TypeScript są bardzo pomocne w sytuacjach, gdy chcemy tworzyć kod, który jest niezależny od konkretnej struktury danych, ale jednocześnie dostosowuje się do typów, z którymi pracuje. Pozwalają na tworzenie bardziej ogólnych i elastycznych rozwiązań, które można wielokrotnie wykorzystywać dla różnych typów danych, jednocześnie zachowując statyczną kontrolę typów w trakcie kompilacji kodu TypeScript.
