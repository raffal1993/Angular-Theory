Shadow DOM (Document Object Model) to technologia, która umożliwia tworzenie i zarządzanie izolowanymi drzewami DOM wewnątrz elementów HTML. Pozwala na odseparowanie struktury HTML, styli CSS i zachowania JavaScript dla danego elementu, co zapewnia hermetyczność i izolację komponentu.

Główne zalety Shadow DOM:

  1. Izolacja stylów: Shadow DOM pozwala na definiowanie i stosowanie styli CSS, które są ograniczone tylko do danego elementu. Style zdefiniowane wewnątrz Shadow DOM nie mają wpływu na elementy poza nim, a style zewnętrzne nie przenikają do elementów wewnątrz Shadow DOM.

  2. Izolacja struktury: Elementy wewnątrz Shadow DOM są odizolowane od reszty dokumentu. Oznacza to, że selektory CSS zastosowane wewnątrz Shadow DOM nie będą miały wpływu na elementy poza nim, a elementy poza Shadow DOM nie będą miały dostępu do elementów wewnątrz niego.

  3. Encapsulation (enkapsulacja): Shadow DOM umożliwia ukrycie implementacji wewnętrznej komponentu. Kod JavaScript, manipulacja DOM i zdarzenia są ograniczone tylko do Shadow DOM i nie mają wpływu na resztę strony. To pozwala na tworzenie komponentów, które są hermetyczne i niekolidują z innymi elementami na stronie.

  4. Reusable Components (komponenty wielokrotnego użytku): Shadow DOM umożliwia tworzenie komponentów, które mogą być używane wielokrotnie w różnych częściach strony lub w różnych projektach. Komponenty z Shadow DOM są samowystarczalne i niezależne od kontekstu, co ułatwia ich ponowne wykorzystanie.

Przykład użycia Shadow DOM w JavaScript:

javascript
const element = document.getElementById('my-element');
const shadowRoot = element.attachShadow({{'{'}} mode: 'open' });

// Definicja struktury HTML wewnątrz Shadow DOM
shadowRoot.innerHTML = `
  <style>
    /* Style dla elementu wewnątrz Shadow DOM */
    :host {{'{'}}
      display: block;
      background-color: #f1f1f1;
      padding: 10px;
    }
  </style>
  <div>
    <!-- Zawartość elementu wewnątrz Shadow DOM -->
    <h2>Hello Shadow DOM!</h2>
    <p>This content is inside the Shadow DOM.</p>
  </div>
`;

W powyższym przykładzie element o id "my-element" ma dołączony Shadow DOM, w którym definiowane są style CSS i struktura HTML. Style są ograniczone tylko do elementu wewnątrz Shadow DOM, a struktura HTML jest izolowana od reszty strony.

Shadow DOM jest szeroko stosowany w tworzeniu komponentów webowych, w tym w technologii Web Components, gdzie umożliwia tworzenie hermetycznych i wielokrotnie używalnych komponentów interfejsu użytkownika.
