Flexbox, znany również jako Flexible Box Layout, to moduł CSS, który umożliwia elastyczne układanie elementów w kontenerze. Flexbox został wprowadzony w CSS3 i zapewnia intuicyjne i potężne narzędzia do tworzenia responsywnych układów stron internetowych.

Podstawowe pojęcia związane z Flexbox to:

  1. Kontener (Container):

    • Element, który jest ustawiony jako kontener flexbox.
    • Działa jako rodzic dla elementów flexowych.
    • Aby zdefiniować kontener flexbox, należy ustawić na nim właściwość display na wartość flex lub inline-flex.

  2. Elementy flexowe (Flex Items):

    • Elementy, które są bezpośrednio zagnieżdżone w kontenerze flexbox.
    • Są to elementy, które zostaną ułożone wewnątrz kontenera flexbox.

Podstawowe właściwości flexbox to:

  1. display:

    • Właściwość display z wartością flex lub inline-flex ustawia element jako kontener flexbox.

  2. flex-direction:

    • Określa kierunek rozmieszczenia elementów w kontenerze flexbox.
    • Może mieć wartości: row (domyślnie), row-reverse, column lub column-reverse.

  3. justify-content:

    • Kontroluje rozmieszczenie elementów wzdłuż głównej osi kontenera flexbox.
    • Określa, jak elementy są zarówno wyrównane, jak i rozmieszczone w kontenerze flexbox.
    • Może mieć wartości: flex-start, flex-end, center, space-between, space-around lub space-evenly.

  4. align-items:

    • Kontroluje wyrównanie elementów wzdłuż drugiej osi kontenera flexbox.
    • Określa, jak elementy są wyrównane w kontenerze flexbox w pionowym kierunku.
    • Może mieć wartości: flex-start, flex-end, center, baseline lub stretch.

  5. flex-grow:

    • Określa, jak elementy flexowe mają rosnąć, jeśli istnieje dodatkowa dostępna przestrzeń w kontenerze.
    • Domyślnie wartość to 0, co oznacza, że elementy flexowe nie będą się rozciągać.

  6. flex-shrink:

    • Określa, jak elementy flexowe mają się kurczyć, jeśli nie ma wystarczającej przestrzeni w kontenerze.
    • Domyślnie wartość to 1, co oznacza, że elementy flexowe będą się kurczyć, aby dopasować się do kontenera.

Flexbox oferuje również wiele innych właściwości i skrótów, takich jak flex-wrap, align-content, flex-basis, order itp., które pozwalają na jeszcze większą kontrolę nad układem elementów.

Przykład użycia Flexbox w CSS:

css
.container {{'{'}}
  display: flex;
  justify-content: center;
  align-items: center;
}

.item {{'{'}}
  flex: 1;
  margin: 10px;
}

W tym przykładzie .container jest ustawiony jako kontener flexbox, a .item są elementami flexowymi. Elementy flexowe są równomiernie rozłożone w kontenerze, a ich marginesy wynoszą 10 pikseli.
