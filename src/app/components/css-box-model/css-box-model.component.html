Model pudełkowy (box model) w CSS opisuje sposób, w jaki elementy HTML są renderowane jako prostokątne pudełka i jakie mają one właściwości dotyczące rozmiaru, marginesów, obramowania i wypełnienia. Składa się z czterech głównych części: marginesu (margin), obramowania (border), wypełnienia (padding) i zawartości (content).

Oto opis poszczególnych części modelu pudełkowego:

  1. Margines (Margin):

    • Margines to odstęp znajdujący się na zewnątrz obramowania elementu.
    • Może być ustawiony jako margines zewnętrzny (external margin) lub margines wewnętrzny (internal margin).
    • Marginesy mogą wpływać na odległość między sąsiadującymi elementami.

  2. Obramowanie (Border):

    • Obramowanie to linia, która otacza zawartość i wypełnienie elementu.
    • Może mieć różne grubości, style i kolory.
    • Obramowanie jest opcjonalne i nie wpływa na rozmiar elementu, chyba że użyto właściwości box-sizing.

  3. Wypełnienie (Padding):

    • Wypełnienie to odstęp między zawartością elementu a jego obramowaniem.
    • Może być ustawione jako wypełnienie zewnętrzne (external padding) lub wypełnienie wewnętrzne (internal padding).
    • Wypełnienie wpływa na rozmiar elementu, ale nie jest uwzględniane w jego szerokości i wysokości.

  4. Zawartość (Content):

    • Zawartość to faktyczna treść lub elementy znajdujące się wewnątrz elementu.
    • Zawartość może mieć szerokość i wysokość, które są mierzone w pikselach lub innych jednostkach, w zależności od ustawień.

Warto również wspomnieć o właściwości box-sizing, która kontroluje sposób obliczania rozmiarów elementów w modelu pudełkowym. Domyślnie właściwość box-sizing ma wartość content-box, co oznacza, że szerokość i wysokość elementu obejmują tylko zawartość. Jednak można ustawić wartość border-box, aby szerokość i wysokość elementu obejmowały również obramowanie i wypełnienie, co ułatwia kontrolę rozmiaru elementów.

Przykład użycia modelu pudełkowego w CSS:

css
div {{'{'}}
  width: 200px;
  height: 100px;
  margin: 10px;
  border: 2px solid black;
  padding: 20px;
  box-sizing: border-box;
}

W tym przykładzie element <div> ma szerokość 200 pikseli (obejmującą zawartość, wypełnienie, obramowanie i margines) i wysokość 100 pikseli. Margines
