Typy narzędziowe (Utility types) w TypeScript to zestaw wbudowanych typów, które ułatwiają manipulację i transformację typów danych. Są to gotowe narzędzia, które można wykorzystać do definiowania nowych typów na podstawie istniejących typów lub do wykonania różnych operacji na typach.

Oto kilka popularnych typów narzędziowych w TypeScript:

  1. Partial<T>: Tworzy typ, który czyni wszystkie właściwości typu T opcjonalnymi. 
typescript
interface Person {{'{'}}
        name: string;
        age: number;
      }
      
      const partialPerson: Partial<Person> = {{'{'}}}; // Wszystkie właściwości są opcjonalne
  1. Required<T>: Tworzy typ, który czyni wszystkie opcjonalne właściwości typu T wymaganymi. 
typescript
interface Options {{'{'}}
        width?: number;
        height?: number;
      }
      
      const requiredOptions: Required<Options> = {{'{'}}
        width: 100,
        height: 200,
      }; // Wszystkie właściwości są wymagane
  1. Readonly<T>: Tworzy typ, w którym wszystkie właściwości typu T są tylko do odczytu (niezmienne). 
typescript
interface Configuration {{'{'}}
        readonly apiKey: string;
        readonly secretKey: string;
      }
      
      const config: Readonly<Configuration> = {{'{'}}
        apiKey: "abc123",
        secretKey: "xyz789",
      };
      
      // Nie można modyfikować wartości w config
      config.apiKey = "newKey"; // Błąd kompilacji
  1. Pick<T, K>: Tworzy typ, który wybiera tylko określone właściwości z typu T na podstawie zbioru kluczy K. 
typescript
interface Person {{'{'}}
        name: string;
        age: number;
        address: string;
      }
      
      const personNameOnly: Pick<Person, 'name'> = {{'{'}}
        name: 'John',
      }; // Wybrano tylko właściwość 'name'
  1. Omit<T, K>: Tworzy typ, który pomija określone właściwości z typu T na podstawie zbioru kluczy K. 
typescript
interface Person {{'{'}}
        name: string;
        age: number;
        address: string;
      }
      
      const personWithoutAge: Omit<Person, 'age'> = {{'{'}}
        name: 'John',
        address: '123 Main St',
      }; // Pominięto właściwość 'age'

Typy narzędziowe w TypeScript dostarczają wiele innych przydatnych funkcji i operacji na typach, takich jak Record, Exclude, Extract, NonNullable, ReturnType itp. Używanie tych typów narzędziowych ułatwia tworzenie bardziej ekspresywnych i elastycznych definicji typów w TypeScript.
