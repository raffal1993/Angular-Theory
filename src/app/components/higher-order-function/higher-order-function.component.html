Higher-order function (funkcja wyższego rzędu) to funkcja, która przyjmuje jedną lub więcej funkcji jako argumenty lub zwraca funkcję jako wynik. W językach programowania, które obsługują funkcje jako typy danych, takich jak JavaScript, TypeScript czy Python, funkcje wyższego rzędu są powszechnie używane do tworzenia bardziej elastycznego i modularnego kodu.

Oto kilka cech charakterystycznych dla funkcji wyższego rzędu:

  1. Przyjmowanie funkcji jako argumentów: Funkcje wyższego rzędu mogą przyjmować inne funkcje jako argumenty. Przekazanie funkcji jako argumentu pozwala na dostosowanie zachowania funkcji wyższego rzędu w zależności od przekazanej funkcji.

Przykład w języku JavaScript:

javascript
function map(arr, fn) {{'{'}}
        const result = [];
        for (let i = 0; i < arr.length; i++) {{'{'}}
          result.push(fn(arr[i]));
        }
        return result;
      }
      
      function double(x) {{'{'}}
        return x * 2;
      }
      
      const numbers = [1, 2, 3, 4, 5];
      const doubledNumbers = map(numbers, double);
      console.log(doubledNumbers); // Output: [2, 4, 6, 8, 10]

W powyższym przykładzie funkcja map jest funkcją wyższego rzędu, która przyjmuje tablicę arr i funkcję fn. Funkcja fn jest wywoływana dla każdego elementu tablicy i wynik jest zapisywany w nowej tablicy result.

  1. Zwracanie funkcji jako wyniku: Funkcje wyższego rzędu mogą również zwracać inne funkcje jako wynik. Może to być przydatne w przypadku tworzenia funkcji fabrykujących lub dostosowywania funkcji na podstawie warunków.

Przykład w języku JavaScript:

javascript
function createMultiplier(multiplier) {{'{'}}
        return function (x) {{'{'}}
          return x * multiplier;
        };
      }
      
      const double = createMultiplier(2);
      console.log(double(5)); // Output: 10
      
      const triple = createMultiplier(3);
      console.log(triple(5)); // Output: 15

W tym przykładzie funkcja createMultiplier jest funkcją wyższego rzędu, która zwraca funkcję. Funkcja zwrócona przez createMultiplier mnoży argument x przez przekazany mnożnik multiplier.

Funkcje wyższego rzędu są potężnym narzędziem w programowaniu funkcyjnym i umożliwiają tworzenie bardziej elastycznego i modularnego kodu. Dzięki nim można tworzyć funkcje bardziej ogólne, które mogą być wielokrotnie wykorzystywane i dostosowywane do różnych potrzeb.
